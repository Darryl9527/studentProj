# studentProj
ssm学生信息管理系统（源码+数据库+视频+报告）课程设计项目，idea、eclipse双版本，spring、springmvc、mybatis
带报告包运行，增删改查 基于javaweb、ssm、jsp、JavaBean、EL、JSTL、JDBC与MVC开发模式实现程序功能。
运行环境: jdk1.8， tomcat7及以上，mysql5.5-5.7，eclipse或idea均可。
